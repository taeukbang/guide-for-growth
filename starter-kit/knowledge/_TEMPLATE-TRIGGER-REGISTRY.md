## 1. 과거 세션 맥락

- **키워드:** 이전에 논의했던, 지난번에, 그때 결정한, 과거 세션, 이전 결정, 히스토리, 맥락 확인
- **로드:** `sessions/SESSION-INDEX.md`
- **적용 원칙:**
  - SESSION-INDEX의 태그와 현재 주제를 대조하여 관련 세션 식별
  - 관련 세션 발견 시 DECISION-DOC 로드하여 이전 맥락 반영
  - 이전 결정과 충돌하는 방향이 제안되면 이전 결정을 명시적으로 언급하고 변경 이유 확인
